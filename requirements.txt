
Tech Stack & Libraries Used
---------------------------
pandas
pymysql
sqlalchemy
psycopg2


Instructions & Terminal Commands
--------------------------------

Environment Setup

macOS – Terminal Commands
------------------------
Install Homebrew (if not installed):
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install Python (if not installed):
brew install python

Upgrade pip:
python3 -m pip install --upgrade pip


Windows – Command Prompt
-----------------------
Install Python from the official website if not installed.
Make sure to check "Add Python to PATH" during installation.

Upgrade pip:
python -m pip install --upgrade pip


Install Required Python Libraries
--------------------------------
pip install pandas pymysql sqlalchemy psycopg2-binary


Project Workflow
----------------
1. Environment Setup:
   Python environment configured with required libraries.

2. Data Processing:
   Data cleaned and transformed using Pandas.

3. Database Loading:
   Cleaned data loaded into MySQL using SQLAlchemy and PyMySQL.
   (PostgreSQL supported using psycopg2 if required.)

4. SQL Analysis:
   Business insights derived using:
   - GROUP BY
   - Aggregate functions
   - Window functions
   - Date functions

5. Visualization:
   Query results exported and visualized using Tableau to build interactive dashboards.


Notes
-----
- No external APIs were used
- Data was processed locally
- SQL queries were written and executed directly on the database
- Visualizations were created from SQL-driven results
